            # CSS 题库

            # 1. postion 中，relative 和 absolute 分别相对于谁定位

            • relative 其在普通流中的原位
            • absolute 最近一级非 static 的元素
            • fixed 窗口 / frame
            • static （默认）正常文档流
            • sticky 根据正常文档流计算

            # 2. display:none 和 visibility:hidden 区别

            • display:none 不显示，不占地
            • visibility:hidden 不显示，占地

            # 3. link 和 @import 的区别

            • link 是 HTML 标签，@import 是 CSS 提供的
            • link 与页面同时加载，@import 等引用它的 CSS 文件加载完后，它再加载
            • @import 在IE5以上才能识别
            • link 权重高于 @import
            • link 不光加载 CSS

            # 4. 介绍下 box-sizing 属性

            • 控制元素盒模型的解析模式
            • 取值: content-box | border-box
            • content-box 元素整体宽高：border + padding + content的width和height
            • border-box 元素整体宽高宽高：content的width和height，设置的border和padding会占用width和height的空间

            # 5. 选择符有哪些，优先级如何

            • id 选择器
            • class 选择器
            • 标签选择器
            • 后代选择器
            • 子选择器
            • 兄弟选择器
            • * 通配
            • 属性选择器
            • 伪类选择器
