# 5. 固定贴底 footer

将 footer 脱离文档流，固定在窗口下方 给内容区添加 padding-bottom 防止内容最后一部分被 footer 吃了

body {
  padding-bottom: <more_than_footer_height>;
}
footer {
  position: fixed;
  bottom: 0;
}
